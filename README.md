# Combo-Breaker
Combination Breakout board for MacSE/30 / IIsi: VGA + DB15 + Ethernet (Bolle or SEthernet)

CURRENT REVISION DOES NOT WORK WITH BOLLE RISER. DO NOT USE WITH BOLLE RISER WITHOUT REMOVING PINS 4,5
